RESTful Routes

A table of all 7 RESTful Routes

Name Path HTTP Verb Purpose
INDEX /cats GET List all cats
NEW /cats/new GET Display new cat form
CREATE /cats POST Create new cat and then redirect somewhere
SHOW /cats/:id GET Show info about one specific cat
EDIT /cats/:id/edit GET Show form to edit one specific cat
UPDATE /cats/:id PUT Update a specific cat and then redirect somewhere
DELETE /cats/:id DELETE Delete a specific cat and then redirect somewhere
